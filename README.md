*웹사이트 주소 :* [글쓰기 루틴 유형 테스트](https://mbti-writing-routine-test.netlify.app/)

## **Description**
- 책 기획부터 출간까지 전체 프로세스를 따라가는 12가지 질문
- 어울리는 책으로 사용자 유형을 분리해, 16가지 결과로 표시
- 결과 페이지를 카카오톡으로 공유 가능
